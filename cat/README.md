# Cat Asset Layout

This folder contains the cat expression assets used by `whitebox`.

## Structure

- `face/` - base face/body variants
- `eyes/` - eye variants
- `mouth/` - mouth variants
- `ears/` - ear variants
- `icons/` - category icons for face, eyes, mouth, and ears
- `preview/` - full preview/reference images
